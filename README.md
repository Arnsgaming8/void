# Void 🌌

An interactive 3D particle system experience built with Three.js and WebGL.

![Nebula](https://img.shields.io/badge/Three.js-WebGL-blue?style=flat&logo=three.js)

## Features

- 15,000 interactive particles
- Real-time mouse/touch interaction with repulsion physics
- Post-processing bloom effects
- Cosmic color palette with additive blending
- Orbital rotation animation
- Responsive design
- 60 FPS performance

## Usage

Simply open `index.html` in a modern web browser. Move your cursor to interact with the particle system.

## Technologies

- Three.js r160
- WebGL
- Custom GLSL shaders
- Post-processing (Unreal Bloom)
